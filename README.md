# 📖 [Capstone-1-Conference] <a name="about-project"></a>

**[Capstone-1-Conference]** is a web project that displays information about an artificial intelligence (AI) conference. On the homepage, visitors can find details about the conference, including its date, location, and main topics. Information about the speakers participating in the conference, is also provided.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

1. Web browser.
2. Code editor.
3. Git

### Setup

In order to clone this repository to your desired folder, run the commands below on your console:

```sh
    git clone https://github.com/Alejuda/Capstone-1-Conference.git
```

### Install

Install this project running the following commands:

```sh
  npm install
```

### Usage

To run the project, open the index.html file on your web browser or with live-server extension.
